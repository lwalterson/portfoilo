This personal portfolio project is made of react native. It has home, about me, contact me, project navigation, and 3 project pages.   
I structured the files to mimic the actual user navigation to make things easy to find. [App.js](http://App.js) holds all the navigation information, and it is done by using Stack Navigator. The various pages are held in the pages folder, and the project pages are in the projects subfolder. The buttons are all made with Touchable Opacity to allow for styling.  The colors come from the background image on the home page, and any future colors added should be the same.   
The file is deployed to Netfly so no setup aside from the link is needed to run it and the code can be accessed through the public GitHub repository.   
Future plans for this project include, ensuring best practices and fixing any code errors, adding more images, making the text and link styling more varied, including more colors, adding more project pages, and creating a more cohesive visual identity.
